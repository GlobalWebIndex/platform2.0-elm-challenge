## Install & Start

⚠️ Using [Yarn Package Manager](https://yarnpkg.com) is recommended over `npm`.

Start and check the app, if you want

```shell
git clone
cd cat-app
yarn
```

```shell
yarn start
```

Build the production

```shell
yarn build
```
